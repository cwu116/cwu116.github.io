算法day09-栈实现队列,队列实现栈

Created2023-02-23|Updated2023-02-23|算法
|Post View:

栈与队列：

栈的特点：先进后出.

队列的特点：先进先出.

C#中常用的栈与队列的方法：

stack.Push(int x)

stack.Pop()

stack.Peek()

stack.Count

queue.Dequeue()

queue.Enqueue(int x)

queue.Peek()

queue.Count

栈实现队列

题目链接：https://leetcode.cn/problems/implement-queue-using-stacks/

看讲解前思路：

使用两个栈，分为1和2. 放入元素时，往栈1放入元素. 需要查看队列顶端元素或者Pop出队列顶端元素时. 首先判断栈2有没有元素，如果有就做peek或者pop. 如果没有，就把栈1的元素pop进栈2里. 再做peek和pop的操作，这样可以实现队列的效果.

C#代码：

时间复杂度： Pop,Peek : O(N) Push: O(1)

空间复杂度：O(N)

看题解后的思路：

peek可以复用pop函数.

队列实现栈

题目链接：https://leetcode.cn/problems/implement-stack-using-queues/

看讲解前思路：

使用两个队列，队列1与队列2. 其中一个队列不为空时，push的时候只往这个不为空的队列push. 一开始都不为空随便选一个. 然后pop和peek时，只需要将不为空的队列dequeue进为空的另一个队列，直到剩一个元素位置，就可以用返回最后这一个元素.

C#代码：

时间复杂度：Pop,Peek : O(N) Push: O(1)

空间复杂度：O(N)

看题解后的思路:

优化版使用一个队列就可以，将弹出去的元素又加到队列里(除了最后一个元素).
